List of Tags

  1. h : h tag is heading tag from size range of 1 to 6
  2. p : p tag is a paragraph tag , we use this tag to write a paragraph
  3. b : b tag is used to bold the words
  4. br : br tag is known as line break tag , this tag is used to break the line and write in new line
  5. header : haeder tag is used to write the haeding of web page
  6. footer : footer tag is used to write the bottom content of web page
  7. ul : ul tag is used to create an un ordered list of content , it uses "." to make list
  8. li : li tag is used to create list of items
  9. ol : ol tag is used to create ordered list of content
  10. a : anchor tag is used to add links , phone number or email adress
